# healthmonitor-quality-docs
Проект HealthMonitor — это веб-сервис компании SetCom, который собирает и анализирует метрики работы приложений.
Документация данного репозитория предназначена для обеспечения качества сервиса, включая требования, тестирование и процессы расследования инцидентов.
